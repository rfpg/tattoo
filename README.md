# Tattoo Design Pages

This repository contains a small static tattoo-design site with two separate design directions and dedicated reference viewers.

## Pages

- `index.html`
  Main `Bran & Sceolang` design brief. This is the hounds-focused homepage with the primary concept image, placement logic, and links into the related viewers.

- `viewer.html`
  Shared image viewer for the hounds design direction. It includes the main concept image, supporting hound references, and the example tattoo photo.

- `ogham.html`
  Standalone `Ceol + Fuaim` Ogham tattoo design page. This version is stripped back to the Ogham line system and supports line-based mode toggles for `Both`, `Fuaim`, and `Ceol`.

- `ogham-viewer.html`
  Dedicated Ogham reference viewer. It leads with a live animated version of the actual togglable Ogham column, and also includes the source sketch and tattoo example image.

## Usage

This project is fully static and intended to work well on GitHub Pages.

- Open `index.html` for the hounds design
- Open `ogham.html` for the Ogham design
- Use `viewer.html` and `ogham-viewer.html` for the corresponding reference viewers

No build step is required.
